## 🌟 Introduction
This project is designed to modernize the customer interaction and inventory management system for **RK Computers**. It features a dual-chatbot system (Customer & Owner) integrated with a fully functional e-commerce website.

## 🚀 Key Features
- **🌐 Responsive Web Interface**: A sleek, modern website for browsing inventory and services.
- **💬 Smart Customer Assistant**: AI chatbot capable of answering product queries, checking stock, and providing recommendations using the Rasa NLU framework.
- **👔 Owner Dashboard/Bot**: Specialized tools for the computer shop owner to manage data and generate reports.
- **📦 Inventory Management**: Integrated system to track computer components, laptops, and accessories.
- **📄 Professional Documentation**: Complete SRS, Analysis, and Design reports included.

---

## 📁 Project Architecture
The repository is split into four primary modules:

### 1. [Computer shop website](./Computer%20shop%20website/)
The public-facing portal built with **HTML/CSS/JS/PHP**.
- Includes product catalogs, contact forms, and service details.
- Seamless integration with the AI Assistant.

### 2. [Chabot for computer shop customers](./Chabot%20for%20computer%20shop%20customers/)
The core AI engine powered by **Rasa**.
- **NLU & Stories**: Trained to handle a variety of customer intents.
- **Actions**: Custom Python scripts for dynamic responses.
- **Frontend**: A dedicated chat UI for customer interaction.

### 3. [Chatbot for computer shop owner](./Chatbot%20for%20computer%20shop%20owner/)
Back-office utilities for data management.
- Python scripts for database operations (`newDB.py`, `mergedb.py`).
- Jupyter notebooks for data analysis and report generation.

### 4. [Documents](./Documents/)
A full set of academic and professional documentation.
- Project Proposal & SRS.
- Analysis & Design Reports.
- Testing Reports & Final Thesis.
