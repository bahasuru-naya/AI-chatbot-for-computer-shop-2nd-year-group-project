# 🌐 RK Computers E-commerce Website

The frontend web portal for RK Computers, providing a modern interface for customers to browse inventory, learn about services, and access the AI Assistant.

## ✨ Features
- **🏠 Home Page**: Overview of services, seasonal offers, and customer testimonials.
- **📦 Inventory**: Dynamic listing of computer hardware and accessories (`inventory.html`).
- **ℹ️ About & Contact**: Detailed shop history and multiple ways to reach out.
- **💬 Chatbot Integration**: Easy access to the AI Assistant via a dedicated UI link.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3 (Bootstrap 4.x), JavaScript (jQuery).
- **Backend**: PHP (used for form processing and dynamic content).
- **Styling**: Custom CSS in `css/style.css` and `css/styleNew.css`.
- **Assets**: Optimized images in `images/` and inventory-specific images in `images inventory/`.

## 🚀 Local Deployment
1. Move the contents of this folder to your web server's root (e.g., `C:\xampp\htdocs\rk-computers`).
2. Start the **Apache** module in XAMPP/WAMP.
3. Open your browser and navigate to `http://localhost/rk-computers/index.html`.

## 📂 Directory Map
- `css/`: Stylesheets for layout and responsiveness.
- `js/`: Scripts for animations, slider effects, and form handling.
- `php/`: Server-side scripts.
- `images/`: General site graphics.
- `images inventory/`: Product-specific images for the catalog.
